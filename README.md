# SRV-INFOTECH-machine-test

### Instalation 
 * Install Node.js , MongoDB on your computer
 * Clone the repository 
 ```
 git clone https://github.com/arjunkv7/SRV-INFOTECH-machine-test.git
 ```
 * Open the file in VS Code
 * Install dependencies
 ```
 npm install
 ```
 * run the program 
 ```
 node bin/www
 ```
